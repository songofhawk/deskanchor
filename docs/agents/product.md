# 产品规格

## 目标

DeskAnchor 是一个低打扰 macOS 菜单栏工具，用于记录显示器组合和窗口布局，并在用户重新连接已知显示器组合后自动恢复窗口位置。

## MVP 范围

- 菜单栏常驻。
- 识别显示器数量、名称、硬件信息、分辨率、缩放和虚拟桌面坐标。
- 保存当前标准窗口的位置、尺寸、所属显示器和基础身份信息。
- 当前显示器组合匹配到历史记录时，自动恢复可匹配窗口。
- 手动保存、手动恢复、暂停自动恢复。
- 辅助功能权限引导。
- 本地保存，不上传数据。

## 权限体验

应用必须说明辅助功能权限的用途：读取窗口位置并移动窗口。首版不依赖屏幕录制权限，不读取窗口内容，不截屏。

权限缺失时：

- 菜单栏图标显示警示状态。
- 保存/恢复操作不静默失败。
- 提供“打开辅助功能设置”入口。

## 边界

- 不承诺恢复所有窗口。
- 不自动启动未运行 App。
- 不强制处理全屏、Split View、跨 Space 窗口。
- 不提供云同步、规则引擎或复杂可视化编辑器。

## 成功标准

- 用户在常见双屏/三屏场景中重连显示器后，大多数标准窗口能回到上次位置。
- 空闲时 CPU 接近 0%，菜单栏常驻存在感低。
- 权限和失败原因可解释。
